# Counter Application

This is a simple Counter Application created using HTML, CSS, and JavaScript.

## Features
- Increase button
- Decrease button
- Reset button

## Functionality
- When the **Increase** button is clicked, the counter value increases.
- When the **Decrease** button is clicked, the counter value decreases.
- When the **Reset** button is clicked, the counter value resets to **0**.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the project.
2. Open `index.html` in a browser.
